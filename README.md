# 📦 Envanter Yönetim Sistemi (Inventory Management System)

Bu proje, **C++ (C++/CLI) ve .NET Windows Forms** kullanılarak geliştirilmiş bir masaüstü Envanter Yönetim Sistemi uygulamasıdır. Kullanıcıların ürün stoklarını, veritabanı entegrasyonu ile güvenli ve kolay bir şekilde takip etmelerini sağlamak amacıyla tasarlanmıştır.

- Youtube: [https://www.youtube.com/watch?v=AGP3M1tXoGE](https://www.youtube.com/watch?v=AGP3M1tXoGE)


## ✨ Öne Çıkan Özellikler

* **Kullanıcı Dostu Arayüz:** Windows Forms ile tasarlanmış sade ve anlaşılır grafiksel kullanıcı arayüzü (GUI).
* **Veritabanı Entegrasyonu:** `DatabaseManager` modülü üzerinden güvenli ve modüler veritabanı (CRUD) işlemleri.
* **Ürün ve Stok Yönetimi:** Sisteme yeni envanter ekleme, mevcut ürünleri güncelleme, silme ve stok durumlarını listeleme.
* **Nesne Yönelimli Mimari:** Veritabanı işlemleri ve arayüz (UI) mantığının birbirinden ayrılarak temiz bir C++ kod yapısı sunması.

## 🛠️ Kullanılan Teknolojiler

* **Programlama Dili:** C++ (C++/CLI)
* **Arayüz (UI):** .NET Windows Forms
* **Geliştirme Ortamı:** Visual Studio
* **Veritabanı:** Yerel Veritabanı (`db/` dizininde saklanmaktadır)
